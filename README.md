j-key-pass
==========

Simple password manager

Простой менеджер паролей.
Очень далекий аналог KeePassX.

## Возможности
* Шифрование Advanced Encryption Standard (AES / Rijndael) 128 bits
* Синхронизация с dropbox
* Генерация паролей
